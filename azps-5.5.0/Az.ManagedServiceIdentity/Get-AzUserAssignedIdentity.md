# Get-AzUserAssignedIdentity

## SYNOPSIS
Возвращает идентификаторы и удостоверения, которые назначены пользователям.

## СИНТАКСИС

### SuscriptionParameterSet (по умолчанию)
```
Get-AzUserAssignedIdentity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupParameterSet
```
Get-AzUserAssignedIdentity -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Сущность **Get-AzUserAssignedIdentity** получает существующие удостоверения, которые назначены пользователям.
