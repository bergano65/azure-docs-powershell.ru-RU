# ConvertTo-AzVMManagedDisk

## SYNOPSIS
Преобразует виртуальную машину с дисками на основе BLOB-дисков в виртуальную машину с управляемыми дисками.

## СИНТАКСИС

```
ConvertTo-AzVMManagedDisk [-ResourceGroupName] <String> [-VMName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **ConvertTo-AzVMManagedDisk** преобразует виртуальную машину с дисками на основе BLOB-дисков в виртуальную машину с управляемыми дисками.
Перед тем как делать это, нужно прекратить работу виртуальной машины.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> ConvertTo-AzVMManagedDisk -ResourceGroupName 'ResourceGroup01' -VMName 'VM01'
```

Эта команда преобразует диски виртуальной машины с именем "VM01" в группе ресурсов "Группа ресурсов" в управляемые диски.
