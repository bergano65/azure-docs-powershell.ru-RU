# Get-AzNatGateway

## SYNOPSIS
Возвращает ресурс Nat Gateway в группе ресурсов по имени или NatGateway Id или все ресурсы nat Gateway в группе ресурсов.

## СИНТАКСИС

### ListParameterSet (по умолчанию)
```
Get-AzNatGateway [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByNameParameterSet
```
Get-AzNatGateway -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzNatGateway -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Возвращает ресурс Nat Gateway в группе ресурсов по имени ИЛИ NatGateway Id OR всем ресурсам nat Gateway в группе ресурсов.
