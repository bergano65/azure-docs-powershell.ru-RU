# Модуль AzureRM. Subscription. Preview
## Nописание
В этом разделе приведены командлеты Azure PowerShell для подписки Azure в платформе диспетчера ресурсов Azure (ARM). Командлеты существуют в пространстве имен Microsoft. Azure. Commands. Subscription.

## Командлеты AzureRM. Subscription
### [Get-AzureRmSubscriptionDefinition](Get-AzureRmSubscriptionDefinition.md)
Получает определение подписки.

### [New-AzureRmSubscriptionDefinition](New-AzureRmSubscriptionDefinition.md)
Создание определения подписки.
