# Модуль AzureRM. Tagss
## Nописание
В этом разделе отображаются разделы справки по командлетам тега Azure.

## Командлеты AzureRM. Tags
### [Get-AzureRmTag](Get-AzureRmTag.md)
Получает предопределенные Теги Azure.

### [New-AzureRmTag](New-AzureRmTag.md)
Создает предопределенный тег Azure или добавляет значения в существующий тег.

### [Remove-AzureRmTag](Remove-AzureRmTag.md)
Удаляет предопределенные теги или значения Azure.
