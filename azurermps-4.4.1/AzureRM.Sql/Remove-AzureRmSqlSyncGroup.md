# Remove-AzureRmSqlSyncGroup

## КРАТКИй обзор
Удаляет группу синхронизации базы данных SQL Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmSqlSyncGroup [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmSqlSyncGroup** удаляет группу синхронизации базы данных SQL Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление группы синхронизации
```
PS C:\>Remove-AzureRmSqlSyncGroup -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -Name "syncGroup01"
```

Эта команда удаляет группу синхронизации базы данных SQL Azure с именем syncGroup01.
