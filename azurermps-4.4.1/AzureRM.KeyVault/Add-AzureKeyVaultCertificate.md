# Add-AzureKeyVaultCertificate

## КРАТКИй обзор
Добавляет сертификат в хранилище ключей.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Add-AzureKeyVaultCertificate [-VaultName] <String> [-Name] <String>
 [[-CertificatePolicy] <KeyVaultCertificatePolicy>] [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Add-AzureKeyVaultCertificate** запускает процесс регистрации сертификата в хранилище ключей Azure Key Vault.

Вторая команда использует **Add-AzureKeyVaultCertificate** , чтобы запустить процесс для создания сертификата.

Третья команда использует командлет Get-AzureKeyVaultCertificateOperation для опроса операции, чтобы убедиться, что она завершена.

В последней команде для получения сертификата используется командлет Get-AzureKeyVaultCertificate.
