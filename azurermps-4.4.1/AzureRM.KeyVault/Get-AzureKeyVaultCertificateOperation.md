# Get-AzureKeyVaultCertificateOperation

## КРАТКИй обзор
Возвращает состояние операции с сертификатом.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureKeyVaultCertificateOperation** получает состояние операции с сертификатом.
