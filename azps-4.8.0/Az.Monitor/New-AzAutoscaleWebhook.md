# New-AzAutoscaleWebhook

## КРАТКИй обзор
Создание веб-перехватчика автомасштабирования.

## Максимальное

```
New-AzAutoscaleWebhook [-ServiceUri] <String> [[-Property] <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzAutoscaleWebhook** создает интерфейс автомасштабирования.

## ИЛЛЮСТРИРУЮТ

### Пример 1

Создание веб-перехватчика автомасштабирования. (автоматически сформированные)

```powershell <!-- Aladdin Generated Example --> 
New-AzAutoscaleWebhook -ServiceUri 'https://contoso.com'
```
