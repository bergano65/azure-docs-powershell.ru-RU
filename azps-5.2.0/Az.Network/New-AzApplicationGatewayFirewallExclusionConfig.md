# New-AzApplicationGatewayFirewallExclusionConfig

## SYNOPSIS
Создание списка правил исключения для шлюза приложения waf

## СИНТАКСИС

```
New-AzApplicationGatewayFirewallExclusionConfig -Variable <String> -Operator <String> -Selector <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Новый список правил исключения для шлюза приложения waf — **New-AzApplicationGatewayFirewallExclusionConfig.**

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> $exclusion1 = New-AzApplicationGatewayFirewallExclusionConfig -Variable "RequestHeaderNames" -Operator "StartsWith" -Selector "xyz"
```

Эта команда создает новое правило исключения для переменной RequestHeaderNames и оператора StartsWith и Selector с именем xyz. Конфигурация списка исключений будет сохранена в $exclusion 1.
