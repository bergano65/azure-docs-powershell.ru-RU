# Set-AzRelayHybridConnection

## SYNOPSIS
Обновляет описание гибридной связи в указанном пространстве имен ретрансляции.

## СИНТАКСИС

### HybridConnectionInputObjectSet
```
Set-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-InputObject <PSHybridConnectionAttributes>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### HybridConnectionPropertiesSet
```
Set-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-UserMetadata <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Командлет Set-AzRelayHybridConnection обновляет описание гибридного подключения в указанном пространстве имен ретрансляторов.
