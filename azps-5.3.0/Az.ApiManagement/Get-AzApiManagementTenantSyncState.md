# Get-AzApiManagementTenantSyncState

## SYNOPSIS
Возвращает состояние последней синхронизации между базой данных конфигурации и репозиторием Git.

## СИНТАКСИС

```
Get-AzApiManagementTenantSyncState -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Командалет **Get-AzApiManagementTenantSyncState** получает состояние последней синхронизации между базой данных конфигурации и репозиторием Git.

## ПРИМЕРЫ

### Пример 1. Просмотр состояния последней синхронизации
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantSyncState -Context $apimContext
```

Эта команда получает состояние последней синхронизации между базой данных конфигурации и репозиторием Git.
