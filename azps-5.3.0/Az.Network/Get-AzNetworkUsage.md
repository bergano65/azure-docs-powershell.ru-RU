# Get-AzNetworkUsage

## SYNOPSIS
Список использования сети для подписки

## СИНТАКСИС

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С Get-AzNetworkUsage и текущим использованием сетевых ресурсов.
