# Get-AzStorageAccountManagementPolicy

## SYNOPSIS
Получает политику управления учетной записью хранилища Azure.

## СИНТАКСИС

### AccountName (по умолчанию)
```
Get-AzStorageAccountManagementPolicy [-ResourceGroupName] <String> [-StorageAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### AccountResourceId
```
Get-AzStorageAccountManagementPolicy [-StorageAccountResourceId] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### AccountObject
```
Get-AzStorageAccountManagementPolicy -StorageAccount <PSStorageAccount>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для выполнения политики управления учетной записи хранилища Azure возвращается поле **Get-AzStorageAccountManagementPolicy.**
