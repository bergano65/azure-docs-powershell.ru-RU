# Get-AzEnvironment

## SYNOPSIS
Получите конечные точки и метаданные для экземпляра служб Azure.

## СИНТАКСИС

```
Get-AzEnvironment [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Новый Get-AzEnvironment получает конечные точки и метаданные для экземпляра служб Azure.

## ПРИМЕРЫ

### Пример 1. Получение среды AzureCloud
```
PS C:\> Get-AzEnvironment AzureCloud

Name       Resource Manager Url          ActiveDirectory Authority
----       --------------------          -------------------------
AzureCloud https://management.azure.com/ https://login.microsoftonline.com/
```

В этом примере показано, как получить конечные точки и метаданные для среды AzureCloud (по умолчанию).
