# AZ. DevSpaces Module
## Nописание
Команды для взаимодействия с контроллером Azure DevSpaces.

## Командлеты AZ. DevSpaces
### [Get-AzDevSpacesController](Get-AzDevSpacesController.md)
Получить или просмотреть контроллер DevSpaces Azure.

### [New-AzDevSpacesController](New-AzDevSpacesController.md)
Создайте новый контроллер DevSpaces Azure.

### [Remove-AzDevSpacesController](Remove-AzDevSpacesController.md)
Удалите контроллер DevSpaces.

### [Update-AzDevSpacesController](Update-AzDevSpacesController.md)
Обновите контроллер DevSpaces, чтобы добавить теги.
