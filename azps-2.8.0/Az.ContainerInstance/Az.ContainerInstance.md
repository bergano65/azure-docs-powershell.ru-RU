# AZ. ContainerInstance Module
## Nописание
В этом разделе отображаются разделы справки для командлетов экземпляров контейнера Azure.

## Командлеты AZ. ContainerInstance
### [Get-AzContainerGroup](Get-AzContainerGroup.md)
Возвращает группы контейнеров.

### [Get-AzContainerInstanceLog](Get-AzContainerInstanceLog.md)
Получение журналов экземпляра контейнера в группе контейнера.

### [New-AzContainerGroup](New-AzContainerGroup.md)
Создание группы контейнеров.

### [Remove-AzContainerGroup](Remove-AzContainerGroup.md)
Удаляет группу контейнера.
