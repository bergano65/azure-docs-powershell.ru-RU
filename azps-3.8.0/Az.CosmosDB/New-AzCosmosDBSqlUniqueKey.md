# New-AzCosmosDBSqlUniqueKey

## КРАТКИй обзор
Создает новый объект CosmosDB SQL UniqueKey.

## Максимальное

```
New-AzCosmosDBSqlUniqueKey -Path <String[]> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzCosmosDBSqlUniqueKey** создает новый объект типа PSUniqueKey.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> New-AzCosmosDBSqlUniqueKey -Path {path}

Path
----
{path}
```
