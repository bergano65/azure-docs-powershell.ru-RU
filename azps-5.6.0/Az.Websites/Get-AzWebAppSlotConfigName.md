# Get-AzWebAppSlotConfigName

## SYNOPSIS
Получить список имен веб-приложений slot Config

## СИНТАКСИС

### S1
```
Get-AzWebAppSlotConfigName [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Get-AzWebAppSlotConfigName [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Get-AzWebAppSlotConfigName** извлекает список имен "Параметры приложения" и "Строка подключения", которые в настоящее время помечены как параметры слота.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\>Get-AzWebAppSlotConfigName -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Эта команда получает строки параметров приложения и подключения, относящиеся к веб-приложению ContosoSite, связанному с группой ресурсов Default-Web-WestUS
