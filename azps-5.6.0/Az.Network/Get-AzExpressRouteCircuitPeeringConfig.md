# Get-AzExpressRouteCircuitPeeringConfig

## SYNOPSIS
Получает конфигурацию пиринга для каналов ExpressRoute.

## СИНТАКСИС

```
Get-AzExpressRouteCircuitPeeringConfig [-Name <String>] -ExpressRouteCircuit <PSExpressRouteCircuit>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzExpressRouteCircuitPeeringConfig** извлекает конфигурацию отношения пиринга для схемы ExpressRoute.

## ПРИМЕРЫ

### Пример 1. Отображение конфигурации пиринга для схемы ExpressRoute
```
$ckt = Get-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $RG
Get-AzExpressRouteCircuitPeeringConfig -Name "AzurePrivatePeering" -ExpressRouteCircuit $ckt
```
